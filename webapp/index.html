시계열 분석 실습 페이지

전통 → ML → DL → 최신(Transformer) 순으로 실습합니다.

일부 모델은 데모 시뮬레이션이며, LSTM은 TF.js로 실제 학습/예측합니다.

ARIMA (데모)

자기회귀 + 이동평균 결합 개념을 간단 예측선으로 시각화합니다.

GARCH (데모)

가격(선)과 변동성(막대)을 함께 보여 변동성 군집을 직관적으로 확인합니다.

XGBoost (데모)

시계열 파생특징의 중요도를 막대그래프로 데모합니다.

LSTM (TF.js 실제 학습)

준비됨.

Transformer (데모/안내)

브라우저에서는 간단 Self-Attention 가중치 데모를 보여주고, 상세 학습은 노트북과 연동합니다.
